# Jornada+

Aplicación web desarrollada para gestionar la jornada laboral y el control de horas.

## Tecnologías

- React
- Vite
- TypeScript
- Fastify
- PostgreSQL
- JWT
- bcrypt
- dotenv

## Funcionalidades

- Autenticación mediante JWT.
- API REST con Fastify.
- Conexión a PostgreSQL.
- Migraciones de base de datos.
- Arquitectura separada entre frontend y backend.

## Instalación

```bash
npm install
npm run dev
